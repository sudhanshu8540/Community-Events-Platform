# 🌍 Community Event Platform

A web-based platform that allows users to create, join, and manage community events. This project helps people connect, participate in local activities, and organize events efficiently.

---

## 🚀 Features

- 📝 Create and manage events
- 🔍 Browse and join events
- 👥 User authentication (Login / Signup)
- 🤖 AI Chatbot support
- 📅 Event categories and filtering
- 🌐 Responsive UI (works on mobile & desktop)

---

## 🛠️ Tech Stack

### Frontend:
- HTML
- CSS
- JavaScript
- (Optional: React.js)
